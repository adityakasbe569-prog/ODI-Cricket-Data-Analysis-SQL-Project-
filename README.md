** ODI Cricket Data Analysis using SQL **

** Overview **
This project analyzes ODI cricket data
using SQL to extract meaningful insights from player
and match statistics.

** Tools Used **
• MySQL
• Excel

** Key Analysis **
• Top run scorers in ODI cricket
•  Highest wicket takers
• Team-wise performance comparison
• Win/loss trends across matches

** Conclusion **
This project demonstrates how SQL can be used to
analyze real-world sports data and generate
clear performance insights.
